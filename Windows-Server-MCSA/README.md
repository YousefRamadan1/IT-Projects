# 🪟 Windows Server Administration Project (MCSA)

A comprehensive Windows Server 2019 administration project 
covering core MCSA topics, implemented in a virtualized lab 
environment at ITI.

---

## 📋 Project Overview

Designed and configured a full Windows Server environment 
including Active Directory, DNS, DHCP, Group Policy, 
Storage Management, and VPN — simulating a real enterprise 
IT infrastructure.

---

## ✅ What Was Implemented

| Topic | Details |
|-------|---------|
| Windows Server Installation | Windows Server 2019 deployment |
| Active Directory (AD DS) | Domain setup, Users, Groups, OUs |
| DHCP | Scope, Reservations, Exclusions |
| DNS | Forward/Reverse Lookup Zones |
| Group Policy (GPO) | Desktop Wallpaper, Logon restrictions |
| Logon Hours | Time-based access restrictions |
| File Sharing | Shared folders with permissions |
| Disk Quota | Storage limits per user |
| Storage Management | Spanned, Striped, Mirrored Volumes |
| Child Domain | Parent-Child domain trust |
| RRAS & VPN | Remote access VPN configuration |

---

## 🛠️ Technologies Used

- Windows Server 2019
- Active Directory Domain Services (AD DS)
- DNS & DHCP Server
- Group Policy Management (GPO)
- Routing and Remote Access (RRAS)
- VMware Workstation (Lab Environment)

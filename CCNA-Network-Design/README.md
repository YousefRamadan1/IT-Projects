# 🌐 CCNA Network Design Project

A complete network design and implementation project built 
using Cisco Packet Tracer, covering core CCNA routing and 
switching concepts at ITI.

---

## 📋 Project Overview

Designed and configured a multi-layer network topology 
simulating a real enterprise network environment, including 
routing protocols, VLANs, inter-VLAN routing, and network 
security fundamentals.

---

## ✅ What Was Implemented

| Topic | Details |
|-------|---------|
| Network Topology | Multi-router & multi-switch design |
| VLANs | VLAN creation and assignment |
| Inter-VLAN Routing | Router-on-a-stick configuration |
| Routing Protocols | Static routing & OSPF |
| Switch Configuration | Trunk & Access ports |
| IP Addressing | Subnetting & IP planning |
| ACLs | Access Control Lists for security |
| DHCP | Dynamic IP assignment |
| Network Security | Port security on switches |

---

## 🛠️ Technologies Used

- Cisco Packet Tracer
- Cisco Routers & Switches (IOS)
- OSPF Routing Protocol
- VLANs & Trunking (802.1Q)
